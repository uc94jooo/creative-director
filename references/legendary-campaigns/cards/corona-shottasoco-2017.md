# ShottaSoCo (Corona · 2017)

## Insight
Fourth entry in the Corona absurdist series. The brand's laid-back identity is elastic enough to absorb any incongruous element — here the collision is with Southern hip-hop aesthetics — and remain recognisably Corona.

## Mechanic
Film applying Corona's visual language to Southern hip-hop references, creating a comedic displacement of the brand's beach mythology into rap culture territory. Part of the same absurdist engine as Beach, Shampoo, and Karate.

## Why it worked
The engine remains consistent: Corona sensory world + maximum incongruity = brand-relevant comedy. The hip-hop displacement works because it is culturally legible, the contrast is sharp, and the brand's relaxed confidence makes it charming rather than appropriative.

## Steal
When building a series, vary the incongruent element — the engine stays the same but the displacement target changes each time. This generates novelty within a consistent creative system.
